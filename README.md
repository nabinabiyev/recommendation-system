# recommendation-system
Creating recommendation system using my own unique algorithm.

This recommendation system has been created by me for Shop.Az dataset.Although my own algorithm can not be as efficient as deep learning methods,but it's surely a great fit for datasets,that do not contain IDs of their users.You are able to see my preprocessing and algorithm creating steps.

You can deploy this algorithm to your organisation,use it for your individial project or stack my algorithm with another one to create another new algorithm.
